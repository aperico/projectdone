# projectdone


Provides a graphical environment which supports and guides the development of a given project.


#Dependencies
* On Fedora:
  pywebkitgtk
    -> sudo yum install pywebkitgtk
* On Ubuntu:
  python-webkit
    -> apt-get install python-webkit
